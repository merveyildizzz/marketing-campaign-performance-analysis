# Marketing Campaign Performance Analysis

SQL and BigQuery project focused on campaign performance, funnel conversion, and traffic source analysis.

## Tools Used
- PostgreSQL
- BigQuery
- Google Analytics 4
- SQL

## Project Scope
- Facebook & Google Ads spend analysis
- ROMI performance comparison
- Funnel conversion metrics
- Traffic source performance
- User journey analysis
